Introduction

This page is intended to be a quick reference for people who would like to use coffeescript with canvas html elements. What sets it apart is that all of the examples are editable. That's right, if you change the code examples they are immediately compiled from coffeescript into javascript and executed on the associated canvas element. Errors are shown above the code area. The id of the canvas element is shown in a blue box above the element.

Text

The canvas allows you to render text to it line by line. There are many settings that can be applied to the text.

context.font = "size font style, font default"

The font

context.strokeText(text, x, y [, maxWidth])

The text parameter is a string (which can include unicode characters). The x,y coordinates are the alignment point of the text (which is the bottom left corner by default). The text will be scaled to be no wider than maxWidth if it is provided. The text is rendered with the current stroke style and font.

-
#canvas-stroke-text

context.fillText(text, x, y [, maxWidth])

The text parameter is a string (which can include unicode characters). The x,y coordinates are the alignment point of the text (which is the bottom left corner by default). The text will be scaled to be no wider than maxWidth if it is provided. The text is rendered with the current fill style and font.

-
#canvas-fill-text

context.measureText(text)

The measureText function takes a string and returns a TextMetrics object. In this example it is using the width to calculate a rectangle as wide as the given text.
